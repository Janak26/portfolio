      <h2 style="font-size: 1.8em">Trade Canada</h2>

      <a href="https://github.com/Janak26/TradeCanada">
        <img src="https://img.shields.io/badge/GitHub-View_on_GitHub-blue?logo=GitHub" alt="View on GitHub">
      </a>

        <h3>An interactive Power BI dashboard to get a basic overview of Canada's trade with its major trading partners.
        </h3>
        <p>Collected data from Statistics Canada Open Data.</p>
        <p>Used Python for cleaning and inserting the data to local PostgreSQL server.</p>
        <p>Used Power BI data extraction query to import data from PostgreSQL.</p>
        <p>Developed DAX queries using CALCULATE, FILTER, and other commom aggregation functions.</p>

        <h3>The below example shows province wise trade proportion. Seleting provinces allows us to visualize its imports and exports on a line chart, as well as its year-on-year changes in imports, exports and trade</h3>
        <p><img src="https://raw.githubusercontent.com/Janak26/portfolio/refs/heads/main/images/TradeImageOne.jpg" alt="Description of the image" width="700" height="395"></p>

        <br>

        <h3>In the below example, seleting a country allows us to visualize the imports and exports from and to that country on a line chart, as well as its year-on-year changes in imports, exports and trade</h3>

      <h2 style="font-size: 1.8em">Gold Price Tracker</h2>

      <a href="https://github.com/Janak26/GoldPriceTrack">
				<img src="https://img.shields.io/badge/GitHub-View_on_GitHub-blue?logo=GitHub" alt="View on GitHub">
			</a>

			<h3>A simple interactive Power BI dashboard showing the prices of Gold and SnP 500 from the year 1968 to 2025.
			</h3>
			<p>Data obtained using Yahoo Finance Python API and other openly available sources.</p>
			<p>Used Python for cleaning and inserting the data to local PostgreSQL server.</p>
			<p>Used Power BI data extraction query to import data from PostgreSQL.</p>
			<p>Developed DAX queries using CALCULATE, FILTER, and other commom aggregation functions.</p>

			<h3>The below example shows a line chart plotting prices of Gold and SnP 500 from the year 1968 to 2025.</h3>
